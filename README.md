# 血液培養採取判定アプリ

成人の感染疑い患者を対象に、NEWS2、qSOFA、例外病態の有無から血液培養採取の目安を表示する静的Webアプリです。

## 使い方

`index.html` をブラウザで開くと、そのまま利用できます。サーバーやビルドは不要です。

## 判定の概要

- 例外病態あり、NEWS2 5点以上、qSOFA 2点以上、または NEWS2 3-4点かつqSOFA 1点以上：血液培養2セットの採取を推奨
- NEWS2 3-4点、またはqSOFA 1点：臨床所見に基づき採取を検討
- 上記以外：ルーチン採取は原則不要

## 注意

本アプリは院内運用案の支援ツールであり、前向きに検証された標準基準ではありません。臨床判断を代替しないため、患者背景と担当医判断を優先してください。

NEWS2は通常のSpO2 Scale 1を使用しています。慢性高二酸化炭素血症でSpO2 Scale 2を用いる患者には未対応です。
